Fourth assignment: social contagion

In this fourth assignment you will investigate the social contagion in a network by implementing the networked coordination game as described this week (see the slides of the lecture, starting from number 34)

Part 1: Use synthetic graphs

  1. In the first part of exercise you can use the synthetic graphs available in the networkx library.
  2. In order to see social contagion you need to generate a graph, define the payoff matrix and select the random nodes that start the contagion.
    Since it can take some time to tune the parameters I recommend you to start with small graphs. The videos below show the contagion in the Zachary Karate Club we have discussed in class and you can start with this graph too. Unfortunately, in the videos the shape of the graphs are different due to the function I used to draw them, but I hope you can see that the communities are quite "strong" against social contagion.

    ps. I have also tried with Erdos-Reny random graphs and with some small world graphs and it was interesting (and fun). For producing the videos I have saved an image at each step of the algorithm and then used the ffmpeg Linux package but I am sure you know many other tools.
